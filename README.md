# Dark Survivor

Браузерный roguelite-выживалка для **Яндекс Игр** в духе Vampire Survivors: авто-атака, волны врагов, прокачка по уровням и метапрогрессия между забегами. Полностью на HTML5 Canvas, без внешних фреймворков — только нативный JS и Canvas 2D API.

## Геймплей

- **4 играбельных персонажа** с разной статистикой и визуалом: Рыцарь (танк со щитом), Маг (урон и скорость атаки), Разбойник (скорость и крит), Некромант (регенерация).
- **11 видов оружия**, открываемых по уровню — от стартового Magic Bolt до Lightning Chain, Void Rift, Soul Drain и Arcane Mine. У каждого своя логика урона, AoE и визуальные эффекты.
- **14 типов врагов**, включая элитных и боссов (мини-боссы и финальные боссы), с масштабированием HP/урона по времени забега.
- **4 главы (Chapters)**, разблокирующиеся по таймеру — меняют пул врагов, палитру фона и атмосферу.
- **Мировые события** (Кровавая луна, Нашествие демонов, Замёрзшее время, Тёмная буря, Проклятая земля) — рандомные модификаторы раунда с собственными эффектами и баннером.
- **Система комбо и киллстриков**, критические удары, отбрасывание врагов, заморозка, яд.
- **Метапрогрессия**: кристаллы за забег → постоянные улучшения (урон, XP, HP, скорость) в отдельном магазине, сохраняются между сессиями.
- **Система достижений** и **локальная/облачная таблица рекордов**.
- **Воскрешение за рекламу** — один реролл за игру при просмотре rewarded-видео.

## Технические особенности

- **Адаптивный рендеринг с LOD**: автоматическое определение качества (`high`/`mid`/`low`) по типу устройства и `navigator.hardwareConcurrency`, плюс динамический LOD-уровень в реальном времени по FPS и количеству врагов на экране — лимиты на частицы, свечение, обводку врагов и эффекты подстраиваются на лету.
- **Мобильная адаптация**: виртуальный джойстик и кнопка рывка, принудительный landscape-режим с оверлеем поворота экрана, корректный расчёт `resizeCanvas()` без `position: fixed` на `body` (чтобы не ломался `window.innerHeight` в мобильных браузерах).
- **Кастомный звуковой движок** на Web Audio API — без аудиофайлов, звуки и фоновая музыка генерируются программно с защитой от спама (`cooldown` на каждый звук) и полной приостановкой звука во время рекламы.
- **Локализация (RU/EN)** через объект строк `STRINGS`, язык определяется автоматически по `ysdk.environment.i18n.lang`.
- **Полная интеграция Yandex Games SDK v2**: `LoadingAPI.ready()`, `GameplayAPI.start/stop()` (с очередью вызовов до инициализации SDK), межстраничная и rewarded-реклама, облачные сохранения через `Player.getData/setData`, лидерборды Яндекса с fallback на локальный `localStorage`.
- **Graceful degradation**: игра полностью работает в standalone-режиме (вне Яндекса), если SDK не доступен — все вызовы к нему обёрнуты в try/catch с локальными запасными вариантами.

## Структура проекта

```
.
├── index.html      # разметка, подключение SDK и canvas
├── style.css       # стили загрузочного экрана, оверлея поворота и canvas
├── script.js       # вся игровая логика (~4750 строк)
└── README.md
```

`script.js` организован по секциям: локализация → интеграция Yandex SDK → рендер и адаптация экрана → персонажи и UI выбора героя → качество/LOD → ввод (клавиатура + тач) → камера → главы и мировые события → метапрогрессия и достижения → объекты карты → игрок → враги → оружие и снаряды → частицы/эффекты → погода → HUD/UI экраны → игровой цикл.

## Запуск

Открыть `index.html` в браузере — либо локально через любой статический сервер (`python -m http.server`), либо напрямую.

Для публикации на Яндекс Играх — загрузить `index.html`, `style.css` и `script.js` в одной папке (без изменений путей).

## Управление

| Действие | Десктоп | Мобильный |
|---|---|---|
| Движение | WASD | Виртуальный джойстик |
| Рывок | Space | Кнопка рывка |
| Пауза | P | Тап по экрану |
| Звук | M | Кнопка в UI |
| Меню/рекорды | Enter / L | Тап по UI |

## Стек

Vanilla JavaScript (ES6+), HTML5 Canvas 2D, Web Audio API, Yandex Games SDK v2. Без сборщиков и зависимостей — три файла, готовые к загрузке как есть.
